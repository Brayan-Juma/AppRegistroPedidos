Registro de Pedidos 

Este proyecto es una solución integral para la gestión de ventas y logística de campo, desarrollada para la plataforma Android. La aplicación combina herramientas de captura de datos automatizada con visualización geográfica avanzada.

🚀 Funcionalidades Principales

1. Registro Inteligente de Pedidos

Escaneo QR: Autocompletado instantáneo de datos del cliente (Nombre, Teléfono, Dirección) mediante códigos QR.

Captura Multimedia: Integración con la cámara del dispositivo para adjuntar evidencia fotográfica de la entrega o pedido.

Geolocalización: Captura automática de coordenadas GPS (Latitud y Longitud) en el momento del registro.

2. Panel de Control (Dashboard)

Gestión Local: Listado completo de pedidos almacenados en una base de datos local SQLite.

Sincronización: Botón de sincronización manual para enviar datos locales hacia una API/Nube.

Exportación CSV: Generación automática de reportes en formato .csv guardados directamente en la carpeta de Descargas del dispositivo.

3. Logística y Seguimiento

Mapa de Ruta: Visualización en tiempo real mediante Google Maps API.

Trazado de Trayectoria: Dibujo de una Polyline que conecta todos los puntos de entrega registrados.

Telemetría: Indicador flotante del tiempo estimado y eficiencia del recorrido.

🛠️ Tecnologías Utilizadas

Lenguaje: Java / Kotlin (Android Studio).

Base de Datos: SQLite.

APIs de Google: Google Maps SDK.

Diseño: Material Design 3 con estética Neo-Tactile.
